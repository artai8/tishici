# AI Video Director Agent

A professional tool for transforming creative ideas into structured video storyboards and generative AI prompts using Gemini models.

## Setup on HuggingFace Spaces

1. Create a new Space with Docker SDK
2. Add your `GEMINI_API_KEY` as a secret in Space Settings
3. Push this code to the Space repository

## Local Development

```bash
npm install
GEMINI_API_KEY=your_key_here npm run dev
